# Ingoga Labs — AI Working Agreement

Before changing this repository, every AI agent must read:

- `docs/INGOGA_LABS_CONSTITUTION.md`

That constitution is the project’s source of truth for brand intent, design
language, tokens, content, motion, accessibility, responsiveness, and product
direction.

## Non-negotiable workflow

1. Read the constitution before planning or editing.
2. Preserve the core idea: African-led research that connects human needs,
   terrestrial systems, and frontier technology.
3. Reuse the documented tokens and patterns before introducing new ones.
4. Keep the website responsive from 320px through large desktop screens.
5. Treat motion as storytelling, not decoration, and always support
   `prefers-reduced-motion`.
6. Use original or licensed assets only. Never copy source-site artwork.
7. Run `npm run build` and perform desktop and mobile browser checks after
   meaningful UI changes.
8. If a user request conflicts with the constitution, follow the user’s request
   while updating the constitution when the change represents a lasting new
   direction.
